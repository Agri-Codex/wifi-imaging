# WiFi Imaging

WiFi Imaging is a real-time WiFi visualization and monitoring desktop application built with Python.

## Features

- Real-time WiFi scanning
- RSSI signal monitoring
- Live signal fluctuation tracking
- Heatmap visualization
- Live graph plotting
- CSV export
- Dark theme UI
- Windows and Linux support

## Libraries Used

- pywifi
- tkinter
- matplotlib
- pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python wifi_imaging.py
```
